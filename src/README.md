# Formation Book Index

This index organizes the manuscript source in `chapters/` into a recommended reading sequence.

## Recommended Reading Order

1. [Introduction](../chapters/00-introduction.md)
2. Read Part I through Part IV in numeric order.
3. Revisit chapters based on the formation domain you are facing now: household, school, parenting, adolescence, peer culture, mentorship, technology, institutions, repair, or intergenerational transmission.
